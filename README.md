# local-whisper

A fast, local speech-to-text tool for macOS, powered by [OpenAI Whisper](https://github.com/openai/whisper). No subscriptions, no cloud — just local transcription optimized for Apple Silicon.

## Why

Most Whisper wrappers out there are either closed-source with a price tag or subscription-based. This project is a simple, free, local alternative built for personal use on an M4 Mac.

## Requirements

- macOS (Apple Silicon recommended)
- Python 3.10+

## Disclaimer

This project was **vibe-coded** — built quickly with AI assistance for personal use. It works for me, it might work for you, but don't expect enterprise-grade polish. PRs and issues welcome if you find something worth fixing.
